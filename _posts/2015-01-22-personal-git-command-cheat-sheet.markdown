## Update index with all changes

{% highlight sh %}
git add -A
{% endhighlight %}

## View Git remote URL

{% highlight sh %}
git remote -v
{% endhighlight %}

## Amending commit message

{% highlight sh %}
git commit --amend
git commit --amend -m "New commit message"
{% endhighlight %}
